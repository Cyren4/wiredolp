WireDolphin en quelques minutes

Un logiciel Open Source

Outil privilegie pour tout développeur aguerri nous avons decide de collaborer sur un repository commun sur Github. GitHub est un site web et un service de cloud qui aide les developpeurs à stocker et à gérer leur code, ainsi qu’à suivre et controler les modifications qui lui sont apportées. L’objectif étant de profiter de ce projet pour apprendre les bonnes pratiques du travail en groupe. En rendant notre travail disponible sur Github il sera possible pour de futurs collaborateurs d'implémenter de nouvelles fonctionalité afin d'avoir un produit de meilleur qualité.
